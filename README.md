# Brainf*ck
Brainfuck interpreter written in Rust

## Usage
1. Clone this repo
2. Run project
```
cargo run -- <file_path>
```
###### You can download latest version for unix-like systems from releases

## Features
- Move left
- Move right
- Increment value
- Decrement value
- Loops
- Write
- Read
